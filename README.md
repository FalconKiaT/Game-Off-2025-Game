# Game-Off-2025-Game

Take center stage as a puppet learning how to interact with humans. In three different scenarios apply your waving skills to accurately respond to your human friends, but be aware you will be judged on your performance!

## Controls

**A S**	Left Arm Shoulder Control \
**D F**	Left Arm Elbow Control \
**J K**	Right Shoulder Control \
**L**	Right Elbow Control \
**Mouse	UI** Menu Interact \
**Esc (Hold)**	Close Game
